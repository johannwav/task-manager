# Task Manager

Aplicación web Full Stack desarrollada con PHP, MySQL y JavaScript.

## Características
- Registro e inicio de sesión
- CRUD de tareas
- Prioridades
- Dashboard responsivo

## Tecnologías
- PHP
- MySQL
- JavaScript
- HTML
- CSS

## Instalación

1. Clonar repositorio
2. Importar database/schema.sql
3. Iniciar XAMPP
4. Abrir localhost/task-manager
